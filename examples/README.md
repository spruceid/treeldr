# Examples

## How to run the examples

```
cargo run -- -i examples/diploma.tldr json-schema https://example.com/Diploma
cargo run -- -i examples/simple.tldr json-schema https://example.com/Foo
cargo run -- -i examples/literals.tldr json-schema https://example.com/Type
cargo run -- -i examples/union.tldr json-schema https://example.com/Type
cargo run -- -i examples/intersection.tldr json-schema https://example.com/C
cargo run -- -i examples/restriction.tldr json-schema https://example.com/Restricted
cargo run -- -i examples/follower-vc-demo.tldr json-schema did:web:demo.spruceid.com/Follow
cargo run -- -i examples/verite.tldr json-schema https://verite.id/verite/appendix/schemas/KYCAMLAttestation
```
